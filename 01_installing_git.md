# 1. Installing Git

## Debian/Ubuntu
- Install git by `apt` package manager

  ```bash
  sudo apt-get install -y git
  ```

- Verify git is installed

  ```bash
  git --version
  ```

## RHEL/CentOS

- Install git by `yum` package manager

  ```bash
  sudo yum install -y git
  ```

- Verify git is installed

  ```bash
  git --version
  ```
