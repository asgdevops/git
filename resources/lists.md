## Unordered list

Unordered lists can be created using any of the characters below:

Char|Option
--|--
`-`| First option
`+`| Second option
`*`| Third option

<table>
<tr>
  <td>Format</td>
  <td>Syntax</td>
</tr>
<tr>
  <td>

   - Apple
   - Banana
   - Grape
     - Melon
       - Watermelon

  </td>
  <td>

  ```text
  - Apple
  - Banana
  - Grape
    - Melon
      - Watermelon
  ```

  </td>
</tr>
</table>
